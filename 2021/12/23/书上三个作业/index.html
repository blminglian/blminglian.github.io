书上三个作业

Posted on 2021-12-23

第一个作业

要求：假定给定两个字符串（可以是空串）s1和s2,要将这两个串合并在一起，要求是：s1串的第一个字符与s2串的第一个字符连接在一起，后续以此类推。若一个串的长度大于另一个串的长度，则保留长串的剩余字符，即连接到新串的末尾即可。例如：若s1=’abcd’、s2=’efghi’，则新串为：‘aebfcgdhi’.

s1 = input("请输入一个字符串")
s2 = input("请输入另一个字符串")
i = 0
if(len(s1)>=len(s2)):
for i in range(len(s2)):#输出长度相同的部分
    {
    print(s1[i],end=""):
    print(s2[i],end="")
    }
for i in range(len(s1)-len(s2)):#输出更长的那个多余的部分
    {
        print(s1[len(s2)],end="")
    }
else:

for i in range(len(s1)):
    {
    print(s1[i],end=""):
    print(s2[i],end="")
    }
for i in range(len(s2)-len(s1)):
    {
        print(s2[len(s1)],end="")
    }

第二个作业

要求：对于用户输入的任意字符串，只显示字母和数字

str = input("请输入一个字符串")
def remove(a):
    b=[]
    for i in a:
        if i not in "!@#$%^&*()_+~`<>?,./:;"'{}[]':
            b.append(i)
    print("".join(b))
print(remove(str))

第三个作业

要求：

制作随机验证码，不区分大小写。流程：

1）用户执行程序

2）给用户显示需要输入验证码

3）用户输入的值用户输入的值和显示的值相同时显示正确的信息:否则继续生成验证码等待用户输入

import random
while True:
num = []
for i in range(4):
    choose = random.randint(0, 1)
    if choose == 0:
        temp = random.randint(97, 122)
        num.append(chr(temp))  # 生成小写字母
    else:
        temp = random.randint(65, 90)
        num.append(chr(temp))#生成大写字母
print("请输入验证码：%s" % num)
num = input()
index = 0
for i in range(4):
    if num[i].upper() == num[i].upper():
        index += 1
if index != 4:
    print("登录失败")
else:
    print("登录成功")
    break
